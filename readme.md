# 🚀 Gautam Kumar — Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built with pure **HTML, CSS, and JavaScript**. Designed with a clean light theme, premium typography, and smooth animations.

🌐 **Live Site:** [gautamrai-28.github.io](https://gautamrai-28.github.io)

---

## 📸 Preview

> Hero section with animated avatar, floating stats, and typed role text.  
> Skills grid with real tech logos, proficiency bars, project cards, and more.

---

## ✨ Features

- **Responsive Design** — works perfectly on mobile, tablet, and desktop
- **Smooth Scrolling** — all navigation links scroll smoothly to sections
- **Typed Text Effect** — cycling role titles in the hero section
- **Scroll Reveal Animations** — elements fade in as you scroll
- **Animated Skill Bars** — proficiency bars animate when visible
- **Tech Icon Grid** — real logos for all technologies
- **Project Cards** — with preview, tech stack, GitHub & Live links
- **Certification Cards** — same style as project cards with PDF view links
- **DSA Strip** — GeeksforGeeks & LeetCode profile links
- **Contact Form** — powered by Formspree, delivers to email
- **Resume Download** — one-click PDF download button
- **Back to Top** — button appears after scrolling
- **Cursor Glow** — subtle glow that follows mouse on desktop
- **Active Nav Links** — highlights current section in navbar

## 🚀 Deployment

## 📬 Contact Form Setup

The contact form uses **[Formspree](https://formspree.io)** to deliver messages to email.

To set it up:
1. Sign up at [formspree.io](https://formspree.io)
2. Create a new form and copy your endpoint URL
3. In `index.html`, update the form action:
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

> ⚠️ Formspree only works on a **deployed/live URL** — not on localhost.

## ✏️ Customization

| What to change | Where |
|----------------|-------|
| Your name, bio, links | `index.html` |
| Colors, fonts, spacing | `style.css` — edit CSS variables at the top |
| Animations, form logic | `script.js` |
| Resume file | Replace `GautamResume.pdf` with your file, update `href` in HTML |
| Project screenshots | Replace preview divs with `<img>` tags |
| Certificate PDFs | Update `href` in certification card buttons |
| GFG / LeetCode URLs | Search `YOUR_GFG_USERNAME` and `YOUR_LEETCODE_USERNAME` in `index.html` |
